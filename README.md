# Test Tata

Aplicación Angular para gestionar productos con formulario, listado y acciones básicas de CRUD.

## Tecnologías

- Angular 19
- TypeScript
- RxJS
- Jest

## Requisitos

- Node.js 18 o superior
- npm

## Instalación

```bash
npm install
```

## Ejecutar la aplicación

```bash
ng server
```

La aplicación se abrirá en el navegador en la URL local indicada por Angular.
